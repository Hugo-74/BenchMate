# ⚗️ BenchMate

**BenchMate** is a Progressive Web App (PWA) designed for laboratory professionals — researchers, PhD students, and technicians.

## Features

- 🧪 **Solution calculator** — powders, molar solutions, % dilutions
- 💾 **Saved recipes** — save and reuse your buffer recipes
- 🔬 **Antibody database** — track dilutions, host, target and stock
- 📦 **Stock tracking** — reagent lots with expiry alerts (push notifications)
- 🧫 **Serial dilutions** — graphical visualization + PDF export
- ⚙️ **Conversions** — concentrations, temperature, centrifugation speed

## Plans

- **Free** — 3 recipes, 5 antibodies, 3 lots
- **Pro** — 4€/month or 36€/year — unlimited + cloud sync + push notifications
